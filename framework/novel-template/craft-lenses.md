# Craft lenses — a diagnostic toolkit

*Diagnostic, **not** generative — run them over the material to find blind spots, never to
force-fit. A good lens asks a yes/no/deliberate-choice question; when the answer surprises you,
it found something. Use a few; internalize; discard the rest.*

**Motivation**
1. **Want vs. Need + The Lie** — conscious external *want*; internal *need*; the **Lie** between.
2. **Opposition web** — every major character a distinct stance on the theme (cast = argument).

**Action (scene)**
3. **Does the scene turn a charged value?** (+→– or –→+). If nothing changes, it's exposition.
4. **Causality: "therefore / but," not "and then."** Beats connect by cause, not sequence.
5. **Scene → Sequel rhythm** — action (goal/conflict/setback) then reaction/decision.

**Plot / Pacing**
6. **MICE** — which thread *frames* the book (Milieu / Inquiry / Character / Event)? Open it
   first, close it last; nested threads close inside-out.
7. **Try/fail escalation (yes-but / no-and)** for the middle + a **reader-knowledge map**
   (who knows what, when) for dramatic-irony control. Don't flatline — oscillate hope/dread.
